# Fraud Detection Project

A Java-based fraud detection web application that analyzes transactions and flags suspicious activity using rule-based logic.

## Features

* Detect suspicious transactions
* Rule engine for fraud patterns
* Web interface for testing transactions
* Unit testing included

## Tech Stack

* Java
* Maven
* Spring Boot (if used)
* HTML / CSS / JavaScript

## Project Structure

* `src/main/java` → Backend source code
* `src/main/resources/static` → Frontend files
* `src/test/java` → Unit tests

## How to Run

```bash
mvn spring-boot:run
```

Then open browser at:

```text
http://localhost:8080
```
